Personal Projects

Here are some of the personal projects I've worked on outside of my coursework. These projects reflect my passion for technology and innovation.


Front-End Engineer | Pokémon Website

This project showcases my expertise in front-end development and UX design. The website offers the following features:

  • Random Pokémon Generation: Users can click on the "Generate Pokémon" button to view a random Pokémon image along with its type and abilities.
  • Manual Pokémon Search: Users can enter the name of a specific Pokémon in the search box and click the "Submit" button to view the Pokémon's image, type, and abilities.
  • Random Pokémon Amiibo Generation: Users can click on the "Generate Pokémon Amiibo" button to view a random Pokémon Amiibo image along with its release date from a pool of 12 valid options.

The project also involves optimizing code and image sizes to enhance the website's performance and user experience. The use of a REST API ensures users have access to real-time and comprehensive Pokémon data.

Full Stack Weather Website

This project showcases my proficiency in full-stack development. The website offers users the ability to access current weather and a 5-day forecast. Key features include:

  • Today's Weather and Forecast: Users can retrieve current weather conditions and forecast details.
  • 5-Day Weather Forecast: Users can view the weather forecast for the next 5 days.

Notably, I developed my own API and endpoints to provide real-time weather data. The project involves both front-end and back-end implementation, demonstrating my capability to create a cohesive user experience.

Brick Breaker Game in Java

This project showcases my programming skills in Java and game development. The game features include:

  • Paddle Control: Players control a paddle at the bottom of the screen to bounce a ball upward.
  • Brick Destruction: The goal is to destroy all bricks located at the top of the screen using the bouncing ball.

Through this project, I gained valuable experience in object-oriented programming and interactive application development.
